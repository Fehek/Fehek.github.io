关于我所使用的软件，插件及网站

本文最后更新于：2021年4月13日 中午

怎么将你日常使用的设备环境打造的舒服，或许这篇会有所帮助。

软件

前言：我用软件最大的雷点是内置广告和弹窗，所以有些虽然好用但是不升级为高级会员广告就不会消失的软件不予推荐。为了软件及时更新与不带其他流氓软件，都推荐去官网下。

阅读类

软件 功能
calibre 支持格式最多，但是排版不是很好，主要用途是书籍的格式转换。最大的缺点是更新太频繁了，几乎每次我打开的时候都有新版本，而且官网下载速度很慢，对强迫症极不友好😡
（我最常用的是转换成awz3格式在kindle上看。虽然可以转换成mobi和awz，推荐awz3，现在在Amazon购买的书大部分都是这个格式了，优点主要是丰富内容排版。还有DRM保护，但对于转换来说这不是优点。顺便一提，kindle最好不要用txt，很容易跳行；对pdf的支持也不是很好）
Sumatra PDF PDF阅读器，高级选项是直接用参数设置的，我懒，没有改，默认的布局也挺好看的。
Icecream Ebook Reader 支持ePub，mobi格式。缺点：排版的页边距太宽了且不能调，支持的中文文本字体很少（几乎没有），全是衬线字体。但作为不买会员能享受到的服务还是挺好的（主要是没广告）。怀念edge自带的ePub阅读器啊，微软是专砍好用的功能吗？
Justread ePub阅读器，UWP应用。免费！自带的字体也够用，缺点是对插图排版不好，图片会从中间被裁开。
Thorium ePub阅读器。除了不能做批注和鼠标滚轮翻页外，其它都可以。

图片与视频

软件 功能
Honeyview 图片查看器，比windows自带的图片查看好用多了，特别是阅读长条漫画很好用。注意：最好不要下载便携版，因为默认布局太丑了，配置会改很多，我上次下载了新版本，又把自己的配置照着旧版重新改了一遍，很麻烦。
PotPlayer 视频播放器。其它下载地址
皮肤下载推荐去贴吧，我自用的一直是Modern-X。
foobar2000 很有名的音乐播放器，功能超全。也推荐去贴吧下载，有大佬汉化，也有很多炫酷好看的皮肤，推荐音乐发烧友下载。我已经退坑了，在用自带的groove听歌。
Listen 1 能导入歌单，但歌词不支持翻译，而且没有小窗口，热键也不能设置。但是！能听一些要收费的歌
MKVToolnix 好用的mkv的提取与封装软件，像我一样的小白最简单的用法就是把视频内置的音轨和字幕导出导入。

界面美观

软件 功能
爱壁纸UWP UWP应用。图片比较多，关键是速度很快。
Splash Wallpaper UWP应用。速度有点慢，但其他splash图片应用要么有广告，要么要收费，这个很良心了。
Windows 主题 直接从Microsoft Store里面下载，图片质量很不错的
Fliqlo 简约漂亮的时钟屏保，官网打不开的点这里下载【更新：由于flash不被支持，此软件已不能使用】
TranslucentTB(汉化) UWP应用。可以控制任务栏外观，使其透明化或者其它效果。
Fences 桌面图标整理工具，steam也有卖，但是换设备激活密钥比较麻烦，推荐在官网买

资源管理与传输

软件 功能
Everything 强大的本地搜索工具，用过都说好，与wox搭配使用效果更佳。
Wox 软件快捷启动工具，效率神器，可快速启动、计算、查找等，与everything一起使用。
Advanced IP Scanner 局域网扫描器，不知道或者忘记了家里路由器的用户名和密码，不能用192.168.1.1查询的，这个软件很方便。
FileZilla 免费开源的FTP软件，可用于FTP和SFTP传输，虽然windows资源管理器就可以进行FTP传输，但耐不住它界面好。
PuTTY 是一款集成虚拟终端、系统控制台和网络文件传输为一体的自由及开放源代码的程序。支持多种网络协议，包括SCP，SSH，Telnet，rlogin和原始的套接字连接。它也可以连接到串行端口。
VNC Viewer 远程桌面控制

磁盘相关

软件 功能
SpaceSniffer 计算机磁盘空间分析器。它使用树状图来可视化磁盘使用情况。
SD Card Formatter SD卡格式化工具，有时候windows自带格式化不是很好，这个软件不错。
Rufus 是一个可以帮助格式化和创建可引导U盘的工具，可以制作Windows、Linux、Android-x86 等操作系统的启动盘。
AS SSD benchmark 硬盘速度测试工具
Crystal DiskMark 也是硬盘速度测试工具，可以更详细地设置队列数与线程数。
HD Tune 有检测机械硬盘坏道，温度检测，磁盘表面扫描等功能。pro版挺好用的，下载渠道嘛，网上找一下。
芯片精灵 可以查询U盘的很多参数（实在找不到官网随便在网上下载一个绿色版吧）

系统优化

软件 功能
Dism++ 强大的Windows系统实用工具
Geek Uninstall 卸载工具，可能不是卸载的最干净的，但界面简单漂亮。
Chocolatey 用指令快速安装软件，管理软件
火绒 杀毒软件，要装杀软只推荐这个，要么就别装，用自带的defender。我用它的主要功能不是杀毒，是垃圾清理，强行粉碎文件等功能。最重要的用处是当你下载了破解软件时windows在告诉你之前就自动删除了，去还原很麻烦，这个在它提醒你时选择不处理就行了。【更新：好像也开始自动隔离删除文件了，你没用了😒】
QuickLook UWP应用。在不运行关联程序的情况下，通过敲击空格键来快速预览文件内容。
PowerToys 微软发布的实用工具集，开源免费
RightMenuMgr 右键菜单管理工具
ContextMenuManager 右键菜单管理工具

下载相关

软件 功能
IDM 总所周知的下载管理器，搭配浏览器idm插件使用更好。
唧唧 b站视频下载工具，只要不是番剧，视频和弹幕都能下载。

其他

软件 功能
Foxmail 邮件客户端，延迟时间有点长，界面也不好看，能用。原来用的是YoMail，还可以直接接收Gmail邮件，但已经凉了。（其实Windows自带的“邮件”挺好用的）
Bandizip 美观好用的解压工具，就是添加密码不能加中文，但解压可以正常用。
万彩办公大师 小工具箱，有很多便利的功能。推荐下载，就算现在用不到，说不定以后哪天就有用了。占内存很小，想用哪个功能可以再下。
ABBYY FineReader 说起OCR，不得不提这个软件。但作为个体用户购买有点贵，也没有必要，其实用不到几次。除非是专门从事相关工作，不然还是自行去搜一下破解版吧。
Caesium Image Compressor 压缩图片
取色器 通过鼠标选取屏幕中像素的颜色
Snipaste 默认F1快捷键截图，而且截图时会自动分块，很方便。还有取色功能，可以取代上一个软件。有UWP版本。
Notepad++ 替代Windows自带的记事本使用，把编码方式设置成UTF-8(无BOM)【更新：作者是港独，而且软件上写了，介意的不要用】
Sublime Text 3 轻量级文本编辑器，打开速度很快，就算不是从事CS的，也可以替代Windows自带的notepad使用。（其实我主要是用来写markdown的）
Visual Studio Code 这个还用推荐吗

插件

前言：插件都是基于谷歌浏览器

插件 功能
集装箱 可以使用谷歌的一些基本服务，建议最先安装，之后就可以顺利进入Chrome网上商店安装扩展了。还有网盘助手等一系列功能。
谷歌上网助手 可以访问谷歌google搜索，gmail邮箱，google+等谷歌产品，建议最先安装，和“集装箱”任选一个
CLEAN crxMouse Gestures 必装！！！鼠标手势，用了就回不去了
下载+ 谷歌自带的下载实在不怎么好用，这个可以解决问题，还带有嗅探
Chrono下载管理器 界面美观，功能强大的下载管理器
Disable automatic tab discarding 过一段时间后回到已打开的标签页会重新加载，谷歌自带的设置里也去掉了关掉此功能的选项，这个拓展解决了这个问题
书签侧边栏 管理书签，要设置内容有点多，但设置好了非常好看和便利
Infinity新标签页(Pro) 自定义新标签页，更新标签页壁纸
Dark Reader 网页调成黑色主题，适合于夜间浏览网页
捕捉网页截图 可以以各种方式截取网页
QCODE 二维码 网页生成二维码和短链，方便手机观看(更新：最新版谷歌浏览器已自带二维码功能)
Video Speed Controller 这个真的超有用！随时调视频加速倍速，对于我这样不加速就不想看视频的人简直是救星。但只支持HTML5视频，不过现在大多数都换成了这种格式。今年flash就要停止更新了。
Cursor style 浏览器自定义光标，自带光标挺多的，也可以自己上传。
IDM Integration Module 搭配IDM使用
哔哩哔哩下载助手 下载b站版权受限（能看不能缓存）的番剧视频，一般的视频当然也能下
One Tab Plus:标签效率管理 减少标签混用（更新：最新版浏览器自身已支持标签页分组）
Save to Pocket 收集暂时不想读的网页，有时间了再来慢慢看，优点在于全平台（我建一个暂时收藏夹，所以不怎么用）
货币转换器 随时调用查汇率，在买外国东西时估算价格很方便
划词翻译 一个强大的翻译拓展（更新：我写某个CSS出现了莫名其妙的布局，除此之外，都挺好用的）
沙拉查词 集合了很多词典，界面比较美观，推荐
彩云小译 日语翻译质量很高，啃生肉利器
AdBlock 广告拦截工具，和”Adblock Plus”任选一个安装
Adblock Plus 广告拦截工具，和”Adblock”任选一个安装
Tampermonkey 著名的油猴插件
GitHub Repository Size 可以显示GitHub仓库和文件库的大小
GitZip for github 可以下载GitHub一个项目的单个文件或文件夹
Enhanced GitHub 在GitHub上显示项目大小和下载等功能

前言：脚本都是用油猴安装的。功能也不用多解释吧，大部分名字都已说明了一切

油猴脚本推荐

脚本 功能
右键在新标签中打开图片时显示最优化图像质量
微博浮图
药方(YAWF) 微博排版修改
知乎排版优化
豆瓣资源下载大师
贴吧页面精简
京东自营过滤
拒绝二维码登录
网页限制解除(改) 实用！！！可以复制一些不允许复制的文字
显示当前网站所有可用的UserJS脚本

将油猴脚本同步

  1. 打开油猴，在设置中将”配置模式“改为“高级”
  2. 往下拉，将同步脚本中“启用TESLA”打勾，“同步类型”选择“浏览器同步”
  3. 油猴脚本与你的浏览器同步成功。记得要注册谷歌账号，不然你浏览器本身就没同步脚本当然也不会同步

手机app

Android

前言：手机软件的广告是真的无法完全避免，所以对广告的要求低一点，以实用为主。

APP 功能
夸克浏览器 无广告，可以倍速看视频，强推
安装包越来越大，很臃肿，不是很推荐了。
via浏览器 不到1M，支持安装插件，可以直接在这里下载一些插件。
酷安 应用很全，相当于国内的play商店。有些软件下的评论有惊喜。
静读天下 电子书阅读器，支持多种格式包括epub，pdf，azw3，mobi，手机上只需下载这一个阅读器就够了。
欧路词典 界面简洁，支持自定义导入词典。
一个木函 体积小，很多实用工具的集合
Microsoft Office 新版本已把Word、Excel和PowerPoint合并在这一个软件中，通过OneDrive同步文件，非常好用。（写这个人尽皆知的软件主要是为了批评一下WPS，为什么广告超多还有那么多人在用）
OneDrive 微软推出的网络硬盘及云端服务，免费用户有5G存储空间，就是同步速度有点慢。
ES文件浏览器 文件资源管理器，自带的解压和看图功能很方便。
MX Player 超好用的手机视频播放器
白描 OCR，中文识别率还行，免费用户可以每日识别5次
极度扫描 OCR，最大的优点是支持公式识别
TextGrabber ABBYY出品的手机OCR软件
什么值得买 网购之前先查一下，有很多优惠券，可以看到史低。运气好可以上到神车。

iOS

前言：安卓上有的大部分就不再推荐，主要写一些苹果专有的软件。

APP 功能
‎Cerulean 功能类似与安卓上的“一个木函”，小工具箱，但体积比它大，IOS上现在的“一个木函”都是假的
ES文件浏览器 每次用数据线传输东西，都只能传到几个特定的软件，这个就可以直接被传入。毕竟是文件管理器，也不算什么。关键是它可以直接解压压缩包和看视频，还自带倍速。不用专门再去下载播放器。
奇点 第三方微博客户端，很好用
Alook 6元你买不了吃亏。自带解压功能，支持大部分图书格式。
多看阅读 txt阅读器。因为是一个书城，所以没有纯粹的txt阅读器那么简洁。但是只用书架功能就还可以，开屏无广告。设置齐全但不繁琐，基础功能都有。
txt阅读器 简评
ES文件浏览器 自带的txt阅读器容易出现断页的情况，而且黑色背景下的字体颜色太刺眼。
Anyview 众多人推荐的，UI做的确实漂亮，而且无广告。但是不能滚动阅读，这对我来说是刚需。
简读 又是没有滚动阅读！简约风，无广告。
多多阅读 这是由于一个奇怪理由被我放弃的阅读器——可设置的选项太多了！我并不需要把每个间距都可设置，状态栏可定制！但是自带的几个主题又不太好用。
主要是设置界面做的太不友好了，要定制一个主题，就一刷地让你设置一满页。还全是文字，没有图形示意，看着就头疼。无广告。
掌阅 最大的缺点是滚动得太快了，滑一下页面就向下滚很远，不好操作。有广告。
